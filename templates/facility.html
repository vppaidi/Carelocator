Here are few scenarios to think

Scenario 1: If there are four retail locations and you want to find the optimal one of the last two locations.
Keep 1 for the first two locations and 0 to the last two locations. (Example Column: 1 1 0 0)
P Value in the next step should be 3 in this case. The DSS will help you to find optimal location from the last two locations

Scenario 2: If there are five existing retail locations and you want to remove two of the locations.
Keep 0 for all the five locations. ( Example Column: 0 0 0 0 0)
P Value in the next step should be 3 in this case. The DSS will help you to find two optimal locations
